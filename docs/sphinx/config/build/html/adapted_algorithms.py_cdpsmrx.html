adapted_algorithms.py_cdpsmrx.cdps_rx_risk_adjustment module

adapted_algorithms.py_cdpsmrx.cdps_rx_risk_adjustment.cdps_rx_risk_adjust(df: dask.dataframe.core.DataFrame, lst_diag_col_name='LST_DIAG_CD', lst_ndc_col_name='LST_NDC', score_col_name: str = 'risk') → dask.dataframe.core.DataFrame

Calculate CDPS MRX risk adjustment score. This function expects the input dataframe to be aggregated to patient level, with columns containing comma separated lists of diagnosis codes & NDC codes during the observed time period.

Parameters

  • df (dd.DataFrame) – BENE level Dataframe with diagnosis code & ndc_code list columns

  • lst_diag_col_name (str) – Name of diagnosis code list column. Defaults to LST_DIAG_CD

  • lst_ndc_col_name (str) – Name of NDC code list column. Defaults to LST_NDC_CD

  • score_col_name (str) – CDPS MRX score output column name. Defaults to score

Return type

dd.DataFrame
