adapted_algorithms.py_betos.betos_proc_codes module

betos_proc_codes.py

adapted_algorithms.py_betos.betos_proc_codes.assign_betos_cat(df: dask.dataframe.core.DataFrame, year: int, claim_type: str = 'medicaid', proc_code_prefix: str = 'PRCDR_CD') → dask.dataframe.core.DataFrame

Get claimwise BETOS codes & categories related to CPT procedure codes in claim. Columns in output dataframe:

  • If concat_codes_to_list=True,

    • lst_betos_code - Comma separated BETOS codes

    • lst_betos_cat - Comma separated BETOS cat

  • Else,

    • One boolean column each for all BETOS codes & BETOS categories

Parameters

  • df (dd.DataFrame) – Claim dask dataframe

  • year (int) – Public use file year

  • claim_type (str) – Medicaid or Medicare claim type

  • proc_code_prefix (str) – Column name prefix for procedure code columns

  • concat_codes_to_list (bool) – When true, output betos code & cats as comma separated string columns

Return type

dd.DataFrame
